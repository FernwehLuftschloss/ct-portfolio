Reflection

For this website, I used Visual Studio Code; it’s where I coded, edited files and actually *made* the website. Everything was pushed to GitHub using GitHub Desktop, where there are incremental summaries and save slots that record the progress of this site. HTML is the structure of the website, so it’s all of the content you see on the page. CSS is the presentation; it makes everything look as pretty as possible, and gave (or gives) me the most trouble.

The main, leading issue I had with this project was aligning everything— it seemed that no matter what I did, it wouldn’t look how I wanted it to. In the end, I’d ended up neglecting the website’s responsiveness in favour of the ‘only’ solutions that worked. For almost every time this happened, I’d change the margin width. Turns out, ‘align-self’ works, too. Another challenge was making everything blend in— the gradient from the header image to the background, and the planets to just be planets, not a picture of a planet with a black background that doesn’t match anything. The gradient was fixed with a div, where I matched the colours using a colour picker (thanks, Teach!). I had to edit and remove the backgrounds of the planets in Canva, and replace all of the images with the new, transparent-background planets. This took multiple trials, and lots of moving things around in my ct-portfolio file. I hadn’t named any new planet files properly, so it was exceptionally hard to remember the numbers of all of the planets— something to remember for next time.

I want this website to be something that can change with me as time goes on, something that grows as I do. The profile picture was the worst. It wouldn’t listen, and still won’t. I’m very eager to fix this, as it’s not very responsive and doesn’t look all that great. I’d also like to adjust the typography a little, as it doesn’t completely match. It's more fitting for my initial idea. Originally, I was going to use a medieval Europe theme (A Knight of the Seven Kingdoms really got me good), but I felt that it was too challenging, at the time. I really hope to make that initial idea come true, when I learn and grow more.
